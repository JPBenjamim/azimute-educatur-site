ESTRUTURA PARA IMAGENS CONCEITO DO SITE AZIMUTE

Usar esta pasta para salvar apenas material curado e pronto para uso visual no site.

- hero: imagem principal da pagina/rota
- card: miniatura forte para home, catalogo ou rota
- mosaico: imagens que entram na colagem de abertura
- galeria: imagens individuais para secoes internas

Regra central:
- nao inventar paisagem, monumento, predio, relevo ou patrimonio
- pode tratar cor, contraste, recorte e montagem apenas com base em fotos reais
- evitar arquivos com texto, logo, moldura, print de feed ou arte promocional

# Mapa de Imagens do Site

> Atualizado em 2026-07-14.
> Este arquivo complementa `especificacao-fotos.md`.
> Aqui o foco nao e "o que fotografar", mas sim:
> 1. qual pagina usa qual asset hoje;
> 2. o que ja esta coerente;
> 3. o que esta provisoriamente herdado de outro roteiro;
> 4. qual acao falta para fechar o acabamento visual.

## Como ler

- `OK` = a pagina usa assets proprios e coerentes com o roteiro.
- `PROVISORIA COERENTE` = a imagem nao e exclusiva daquele roteiro, mas conversa com o mesmo territorio/eixo.
- `PROVISORIA HERDADA` = a pagina usa imagem de outro roteiro; nao quebra o site, mas enfraquece a identidade da pagina.
- `AUTORIZACAO` = a foto final depende de permissao da comunidade, instituicao ou operacao local.

## Proximo uso deste arquivo

Este mapa serve para tres coisas:

1. marcar rapidamente o que o site ja tem e o que esta improvisado;
2. alimentar uma planilha externa de producao de fotos;
3. guiar a troca futura dos arquivos em `site/assets/images/cinematic/` sem perder o controle.

Quando uma foto final entrar, atualizar:
- o nome do asset nesta tabela;
- o status da linha;
- e, se necessario, `especificacao-fotos.md`.

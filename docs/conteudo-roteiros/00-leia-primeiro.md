# Pacote de conteúdo por roteiro — leia antes de usar

> Gerado em 2026-07-13. Escopo original: 38 roteiros. Catálogo atual após remoção de duplicados: 36 cards de roteiro, organizados nos mesmos 10 arquivos de rota do site.
> Regra seguida: nunca inventar local, horário, fato histórico ou dado de visitação. Onde a informação não existe em fonte já pesquisada, o campo diz "pendência" em vez de um texto genérico.

## Como usar este pacote

Para cada roteiro, os arquivos de rota trazem os 6 itens pedidos:

1. **Onde é** — cidade/território, paradas reais.
2. **Prólogo local** — resumo histórico/cultural/geográfico.
3. **Jornada do aluno** — 3 a 4 paradas concretas com o que o aluno observa/investiga.
4. **BNCC** — formato `CÓDIGO (Disciplina) — descrição`.
5. **Revisão de copy** — o que está bom, o que precisa mudar, no tom Azimute.
6. **Fotos necessárias** — o que cada foto real precisa mostrar (sem inventar imagem).

Isto é conteúdo, não HTML. Nada aqui deve ser colado direto no site sem revisão — é insumo para quem for editar as páginas.
